# 唐山海钓指南核验与图片方案 v1

## 文档定位

这份文档服务三件事：

1. 建立钓点真实性与时效性核验机制
2. 建立图片搜集与占位机制
3. 对高优先级钓点做第一轮联网核验

当前版本先覆盖：

- 海岸钓前 5
- 出海钓前 3

## 核验总原则

这个网站的底层逻辑不是“只推荐零风险点”，而是“优先推荐真正强的好钓点”。

因此核验不是为了删点，而是为了把信息分层表达清楚：

- 点位本身强不强
- 信息来源稳不稳
- 有没有动态限制
- 有没有生命危险型威胁

## 风险表达总原则

### 1. 普通限制不删点

以下内容不作为删除推荐的理由，只写进第三层级“风险与注意事项”：

- 港区动态管控
- 临时封禁
- 作业区边界
- 进入性不稳定
- 停车限制
- 补给不足

### 2. 只有生命危险型威胁才在第二层级前置

以下类型允许在第二层级卡片前置危险提示：

- 深水无护栏
- 消波块极滑
- 涨潮易被困
- 夜间坠落风险高
- 强风浪冲击明显

## 核验机制

### A 级核验：强核验

必须优先核验的内容：

- 国家级海洋牧场身份
- 重点海域或港区的官方属性
- 与保护区、生态红线、港区管理直接相关的信息
- 容易变化的合规信息
- 容易影响用户是否能去的动态信息

### B 级核验：中核验

建议核验但允许先以“待复核”上线的内容：

- 具体钓段叫法
- 具体几公里区段
- 民间常用叫法
- 地图标注位置
- 博主标注图

### C 级核验：内容型经验信息

可以先按现有资料入库，后续再追加验证：

- 渔获排序
- 汛期强弱
- 钓法适配
- 饵料偏好

### 图片优先级

1. 官方实景图
2. 政府或官方报道图
3. 能明确对应区域的公开环境图
4. 地图标注图
5. 博主标注图
6. 无图时用结构示意图或区域占位图

### 图片使用规则

- 第二层级图片用于“快速识别环境”
- 第三级图片用于“理解点位结构与气质”
- 没有现场图时，允许使用环境图或地图标注图
- 工业港区、防波堤、港池类点位，不强求绝对现场图，但必须保证图与区域基本一致

### 图片落库时必须记录

- 图片来源链接
- 对应钓点
- 图片类型
- 是否精确到现场
- 是否为占位图
