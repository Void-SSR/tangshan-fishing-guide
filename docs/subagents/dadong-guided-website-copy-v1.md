# 大东三层级逐字稿结构建议 v1

## A. 第一层级应该如何说

### 目标

- 第一层级不是展示站首页，而是出钓方式的第一决策入口。
- 它要帮助用户先明确：这次是`海岸钓`还是`出海钓`。
- 它只承担“分流”和“建立可信感”，不承担钓点细节解释。

### 页面顺序

1. 顶部品牌标题  
   - 唐山海钓指南
2. 一句核心说明  
   - 先按出钓方式进入，再看当前模式下最值得去的钓点。
3. 两张主入口卡  
   - 海岸钓  
     - 说明：从防波堤、港池、入海口、闸口里找更强、更稳的钓点  
     - 按钮：进入海岸钓指南
   - 出海钓  
     - 说明：从海洋牧场、人工鱼礁、沉船礁区里找更强、更稳的海域  
     - 按钮：进入出海钓指南
4. 底部辅助信息  
   - 排名说明
   - 合规说明
   - 最近更新

### 文案语气原则

- 要像“先帮用户选路”，不是像“欢迎页”。
- 语气要稳，不喊口号，不堆形容词。
- 首页说明必须服务后续点击动作，不能讲太散。
- 避免把首页写成内容大厅或资讯页。

## B. 第二层级应该如何说

### 目标

- 第二层级是模式内的钓点决策页。
- 它的任务是让用户在`海岸钓`或`出海钓`模式下，快速看清哪里最值得去。
- 它强调“推荐强度排序 + 快速比较 + 进入详情”，不是详情展开页。

### 页面顺序

1. 顶部栏  
   - 返回
   - 当前模式标题：海岸钓 / 出海钓
   - 一句模式说明：按鱼多、稳定、值得去排序
2. 轻量工具区  
   - 搜索
   - 区域筛选
   - 目标鱼筛选
   - 新手友好筛选
   - 排序说明
3. 钓点列表  
   - 每张卡必须包含：
     - 钓点图片
     - 钓点名称
     - 模式标签
     - 区域与导航信息
     - 核心钓段说明
     - 主打鱼种
     - 核心汛期
     - 排名标识
     - 查看详细按钮
4. 第二层级只前置生命危险型提示  
   - 深水无护栏
   - 消波块极滑
   - 涨潮易被困

### 文案语气原则

- 要像“榜单 + 快速判断页”，不是像专题文章。
- 每张卡都要回答：为什么值得点开。
- 句子短，信息密度高，避免大段描述。
- 排名口吻可以保留，但对外表达要稳，例如“高优先级核心点位”，不要满屏“天花板”。

## C. 第三层级应该如何说

### 目标

- 第三层级是单钓点决策页。
- 它的任务是把这个点讲透，让用户决定：去不去、什么时候去、怎么去、怎么钓。
- 它不再混入第二层级列表结构，只服务当前这个钓点。

### 页面顺序

1. 顶部信息  
   - 返回
   - 模式标签
   - 排名标签
   - 钓点名称
2. 主图区  
   - 钓点主图
   - 图片说明
3. 基础信息区  
   - 区域与导航信息
   - 核心钓段说明
   - 主打鱼种
   - 补充鱼种
   - 核心汛期
   - 黄金窗口期
4. 实战信息区  
   - 推荐钓法与线组
   - 推荐饵料
   - 新手友好度
   - 适合谁去
5. 风险与注意事项  
   - 管控信息
   - 进入性
   - 作业区边界
   - 危险提醒
6. 相关推荐  
   - 同区域推荐
   - 同模式推荐
   - 同目标鱼推荐
7. 信息区  
   - 信息来源
   - 更新时间
   - 待复核提示

### 文案语气原则

- 要像“出钓决策卡”，不是像资讯长文。
- 首屏就要告诉用户：这个点为什么值得去、什么时候去。
- 风险写清，但不要把整个页面写成警示公告。
- 推荐区要服务下一步行动，不要写成泛泛而谈的阅读延伸。
