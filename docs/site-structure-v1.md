# 唐山钓鱼指南网站交互结构 v1

## 网站定位

这不是一个单纯“钓点罗列站”，而是一个能帮用户快速做出出钓决策的本地海钓指南站。

网站要解决的核心问题：

- 去哪钓
- 现在适不适合去
- 钓什么
- 怎么钓
- 风险是什么
- 到那以后从哪一段开始

## 用户入口设计

我建议首页第一层只做 3 个主入口：

1. 船钓找点
2. 岸钓找点
3. 按目标鱼找点

这样比一上来铺太多栏目更清晰。

## 一级导航建议

- 首页
- 船钓钓点
- 岸钓钓点
- 目标鱼攻略
- 季节与潮汐
- 新手指南
- 合规与安全
- 最近更新

## 首页结构建议

### 1. Hero 首屏

首屏不要做普通资讯站风格，建议更像“出钓前的导航台”。

建议内容：

- 标题：唐山钓鱼指南
- 副标题：按船钓、岸钓、目标鱼和季节快速找到更合适的点位
- 两个主按钮：进入船钓 / 进入岸钓
- 一个辅助按钮：按海鲈 / 黑头 / 鲅鱼找点

### 2. 快速决策区

这个区域非常关键，建议做成 4 个快速入口：

- 我想船钓
- 我想岸钓
- 我想钓海鲈
- 我是新手

### 3. 热门核心区块

建议首页直接放 6 个主推点，不要太多：

- 船钓 3 个
- 岸钓 3 个

优先使用这批资料里的第一梯队点位。

### 4. 按区域进入

区域卡片建议做 5 个：

- 曹妃甸
- 乐亭
- 丰南
- 京唐港
- 滦南

### 5. 按目标鱼进入

建议单独做一个三列模块：

- 海鲈
- 许氏平鲉（黑头）
- 鲅鱼

每列下面带 2-3 个优先推荐点。

### 6. 安全与合规提醒

首页只放简版提醒，不放大段法规说明：

- 先查天气
- 先看潮汐
- 港区和工业区点位先确认能否进入
- 防波堤必须防滑

## 船钓列表页

这个页面应该是一个强筛选工作台。

建议筛选器：

- 区域
- 钓场类型
- 目标鱼
- 季节
- 新手友好度
- 成本/距离

建议列表卡片信息：

- 点位名称
- 所属区域
- 核心鱼种
- 最佳季节
- 渔获级别
- 新手友好度
- 进详情按钮

## 岸钓列表页

岸钓页的逻辑和船钓页不同，应该更强调“地形结构 + 安全”。

建议筛选器：

- 区域
- 结构类型：防波堤 / 港池 / 入海口 / 盐场闸口
- 目标鱼
- 是否适合夜钓
- 是否适合新手
- 停车便利性

建议卡片重点展示：

- 核心钓段描述
- 涨潮还是退潮更强
- 是否需要防滑钉鞋
- 是否适合单人首次前往

## 钓点详情页结构

每个详情页建议按下面顺序展开：

1. 点位名称与一句话定位
2. 模式标签：船钓 / 岸钓
3. 区域与导航信息
4. 核心钓段说明
5. 主打鱼种
6. 核心汛期与黄金时间
7. 推荐钓法与线组
8. 推荐饵料
9. 新手友好度
10. 风险与注意事项
11. 相关推荐点位
12. 信息来源与更新时间

## 目标鱼攻略页

建议每条核心鱼单独做一页：

- 海鲈
- 黑头
- 鲅鱼

每页内容结构：

1. 这种鱼在唐山的大致分布
2. 船钓优先点
3. 岸钓优先点
4. 更适合的季节
5. 推荐钓法与饵料
6. 新手建议

## 季节与潮汐页

这个页面建议更偏实用，不做花哨图表堆砌。

建议内容：

- 春汛看什么
- 秋汛看什么
- 大潮期怎么选点
- 闸口点位怎么判断窗口
- 岸钓优先时段
- 船钓优先时段

## 新手指南页

建议只回答新手最关心的几个问题：

- 第一次来唐山，先岸钓还是先船钓
- 新手最稳的 5 个点
- 最少要带什么装备
- 哪些地方不要逞强
- 什么天气不要去

## 最近更新页

这个页面对建立网站可信度很重要。

建议展示：

- 新增点位
- 规则修订
- 风险提示修订
- 最近核验时间
- 待复核内容

## 核心交互链路

### 链路 1：新手找点

首页 -> 我是新手 -> 岸钓优先 / 船钓体验 -> 低风险点位列表 -> 点位详情

### 链路 2：按鱼找点

首页 -> 目标鱼 -> 选择海鲈 / 黑头 / 鲅鱼 -> 点位推荐 -> 点位详情

### 链路 3：按地区找点

首页 -> 区域卡片 -> 进入曹妃甸 / 乐亭等区域页 -> 船钓与岸钓分流 -> 点位详情

### 链路 4：临时决策

首页 -> 看季节 / 潮汐 -> 当前窗口推荐 -> 跳转到点位详情

## 视觉表达建议

首页偏氛围，内页偏效率。

建议的界面气质：

- 海港、潮汐、风向、钢铁海岸感
- 深海蓝 + 灰蓝 + 沙色点缀
- 大标题克制，不做浮夸海报字
- 强调层级、留白、信息密度

## 第一版不建议急着做的功能

- 精确地图热区
- 用户评论系统
- 登录系统
- 复杂后台
- 实时天气抓取

这些都可以等内容体系稳定后再加。
