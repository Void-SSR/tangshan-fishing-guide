# 第三层级详情页逐字稿定稿 v1

## 页面定位

第三层级不是资料堆砌页，也不是第二层级的大列表延伸页。

第三层级只做一件事：

- 让用户完成单点决策

用户看完这一页，必须能直接回答四个问题：

1. 值不值得去
2. 什么时候去
3. 怎么去
4. 怎么钓

---

## 页面顺序

1. 顶部信息
2. 标题区
3. 主图区
4. 基础信息区
5. 实战信息区

## 第三层级文案红线

- 不再保留第二层级的大列表壳子
- 不上来直接堆玩法细节
- 不把动态信息写成永久事实
- 不删普通限制，只放在风险区说明
- 不只罗列资料，不给用户形成决策结论

---

## 第三层级交付标准

- 用户看完后能判断值不值得去
- 用户看完后能判断什么时候去
- 用户看完后能判断怎么去、怎么钓
- 用户看完后能明确风险和替代推荐
