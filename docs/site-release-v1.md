# 唐山海钓指南 第一版交付说明 v1

## 版本定义

这是唐山海钓指南的**第一版可交付网站**。

目标不是一次性把所有图片和核验做到终态，而是把“可用、可信、结构稳定”的第一版交付出来。

## 第一版包含什么

- 首页 `/`
- 海岸钓列表 `/shore/`
- 出海钓列表 `/boat/`
- `17` 个单钓点详情页 `/spots/:id/`
- 排名说明 `/about-ranking/`
- 合规说明 `/compliance/`
- 最近更新 `/updates/`
- PWA 基础资源：
  - `manifest.webmanifest`
  - `sw.js`
  - 图标与主屏安装入口

## 第一版已达到的标准

### 产品结构

- 首页只做 `海岸钓 / 出海钓` 分流
- 第二层级只做同模式钓点扫描
- 第三层级只做单点详情
- 不存在同页展开偷换层级

### 内容结构

- 海岸钓 `9` 个点位
- 出海钓 `8` 个点位
- 总计 `17` 个详情页
- 每个详情页都承接：
  - 基础信息
  - 核心钓段
  - 核心汛期
  - 推荐钓法与线组
  - 推荐饵料
  - 风险与注意事项
  - 相关推荐
  - 来源与核验说明

### 交互与体验

- 搜索、筛选、清空筛选
- 列表返回状态恢复
- iPhone 竖屏优先布局
- 第二层级危险提示只服务生命危险型威胁
- 第三级保留核验与图片说明

### 技术状态

- 静态站可构建
- `dist/` 产物已生成
- `test:smoke` 已通过
- 浏览器 E2E 脚本已存在，当前环境受限未作为阻断项

## 第一版已知限制

以下是**已知限制，不阻断第一版交付**：

1. 真实公开图片仍在持续替换中  
   部分点位仍使用公开环境图、地图标注图或结构占位图。

2. 动态进入性与港区管控仍需持续复核  
   这些信息已经按规则留在第三层级，不影响第一版上线。

3. 浏览器动态回归未在当前环境完整实跑  
   当前环境本地端口受限，因此以静态 smoke 作为收口门槛。

4. 当前默认按根路径部署  
   如果后续改成子目录部署，需要单独复核 PWA 的路径配置。

## 当前上线门槛判断

基于当前实现，第一版已经满足可交付条件：

- 大北：无核验阻断项
- 大东：无 UI 阻断项
- 大南：无 QA 阻断项

## 第一版后的优先继续项

1. 继续推进 `P0` 真图替换
2. 环境允许后补跑浏览器 E2E
3. 继续补高优先级点位更贴近现场的公开图
4. 继续把核验状态从 `partially_verified / pending_review` 往上提
