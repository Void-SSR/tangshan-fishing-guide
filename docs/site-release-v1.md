# 唐山海钓指南 第一版交付说明 v1

## 版本定义

这是唐山海钓指南的**第一版可交付网站**。

目标不是一次性把所有图片和核验做到终态，而是把“可用、可信、结构稳定”的第一版交付出来。

当前这一版已经按“指导型网站”重新收束三层级：

- 第一层级：首页只做模式分流
- 第二层级：列表页只做同模式点位比较
- 第三层级：详情页只做单点决策

## 第一版包含什么

- 首页 `/`
- 海岸钓列表 `/shore/`
- 出海钓列表 `/boat/`
- `17` 个单钓点详情页 `/spots/:id/`
- 排名说明 `/about-ranking/`
- 合规说明 `/compliance/`
- 最近更新 `/updates/`
- PWA 基础资源：
  - `manifest.webmanifest`
  - `sw.js`
  - 图标与主屏安装入口

## 第一版已达到的标准

- 首页只做 `海岸钓 / 出海钓` 分流
- 第二层级只做同模式钓点比较
- 第三层级只做单点决策
- 不存在同页展开偷换层级

- 列表页卡片已按“短摘要 -> 区域导航 -> 核心钓段 -> 主打鱼种 -> 核心汛期 -> 简标”重排
- 详情页标题区已改为“一句话判断”，主图说明已改为用户可读说明

## 第一版已知限制

以下是**已知限制，不阻断第一版交付**：

1. 真实公开图片仍在持续替换中  
   部分点位仍使用公开环境图、地图标注图或结构占位图。

2. 动态进入性与港区管控仍需持续复核  
   这些信息已经按规则留在第三层级，不影响第一版上线。

3. 浏览器动态回归未在当前环境完整实跑  
   当前环境本地端口受限，因此以静态 smoke 作为收口门槛。

4. 真实图片仍未完成终态替换  
   当前已具备公开环境图 / 标注图 / 占位图上线能力，但不是终态素材质量。

## 当前上线门槛判断

基于当前实现，第一版已经满足可交付条件：

- 大北：无核验阻断项
- 大东：无 UI 阻断项
- 大南：无 QA 阻断项
- 本地安全底线：无阻断项

当前阶段更准确的定义是：

- 已进入**阶段性正式查验版**
- 后续继续改动时，先过安全预检和大南 Gate，再给用户查验

## 当前正式版本要求

从这一版开始，任何交给用户查验的阶段版本，都必须同时满足：

- 先过 `npm run safety:preflight`
- 再过大南 QA Gate
- 最后再交付用户查验

不满足这三项的版本，不视为正式版本。

## 第一版后的优先继续项

1. 继续推进 `P0` 真图替换
2. 环境允许后补跑浏览器 E2E
3. 继续补高优先级点位更贴近现场的公开图
4. 继续把核验状态从 `partially_verified / pending_review` 往上提
